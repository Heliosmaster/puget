extend-tagged-value

macro

(extend-tagged-value t tag expr)

Extends the TaggedValue protocol with implementations which return the given symbol as the tag and use the given expression to calculate the value. The expression should resolve to a function which accepts one argument and returns the serialized value. This macro also defines a print-method which delegates to edn-str.
